# MNIST-El-Yazi-Rakam-Tanima
Makine Öğrenmesi dersi kapsamında, grup arkadaşım Sena ile birlikte geliştirdiğimiz bu proje; MNIST veri setini kullanarak el yazısı rakamların yüksek doğrulukla tanınmasını amaçlamaktadır. Proje sürecinde veri ön işleme, model mimarisi oluşturma ve başarım metriklerinin analizi gibi uçtan uca makine öğrenmesi adımları uygulanmıştır.
